Detailed Description

----------------------------------------------------------

Lighthouse3D

VSFL - Very Simple Font Library

Full documentation at http://www.lighthouse3d.com/very-simple-libs

This class aims at displaying text strings using bitmap fonts for core versions of OpenGL.

This lib requires:

VSML (http://www.lighthouse3d.com/very-simple-libs)

and the following third party libs:

GLEW (http://glew.sourceforge.net/), TinyXML (http://sourceforge.net/projects/tinyxml/), and DevIL (http://openil.sourceforge.net/)

Note: The font files (xml and tga) are produced by an old software called FontStudio by Michael Pote from Nitrogen ---------------------------------------------------------------

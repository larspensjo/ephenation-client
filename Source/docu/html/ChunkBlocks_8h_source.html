00020 //
00021 // Manage the actual blocks inside a chunk, as well as some information about the chunk as given by the server.
00022 // Chunks from the server, or from the local cache, are compressed.
00023 //
00024 // Jelly blocks is a special mechanism that can temporarily turn any block type into air. This is controlled from
00025 // activator blocks, and it is used for making trap doors and other ways to open a passage. The reason that the
00026 // management of this is in ChunkBlocks is that it will actually change the raw data of the chunk.
00027 //
00028 // TODO: Most data in ChunkBlocks is as decoded from the cache or as received from the server. It should be packged
00029 // in private structure.
00030 //
00031
